tFormer.js

Empower your HTML forms

Created by Vasiliy Zubach / @TjRus

The main idea

tFormer is a JavaScript plugin for validation and basic controlling your HTML forms.

Is this something new?


No...


But...


... this is a new approach.

Distinctive Features

  1. Independence - no jQuery needed;
  2. Short & intuitive validation rules;
  3. Independent validation object;
  4. Changing rules on-the-fly with one method;
  5. Changing options on-the-fly;
  6. Processing, error and disabled classes;
  7. Flexible request validation;

Browser compatibility

  1. Chrome
  2. Safari;
  3. Firefox;
  4. Opera;
  5. ... IE8+; ( who doesn't love it?! =D )

Installation

1. Download the source - tformer.min.js;

2. Add it to your page:

<script src="tFormer.min.js"></script>

3. Define new tFormer and it's rules for your form:

Our form:

<form id="my_form_id">
	<input type="text" name="zip" />
	<input type="text" name="email" />
	<input type="submit" value="Submit" />
</form>

Defining tFormer:

var my_form_id = new tFormer('my_form_id',{
		fields: {
			zip: 'a1 l=5', // define only rules
			email: { //defining rules and other options
				rules: '* @',
				timeout: 500
			}
		}
	});
</script>

Validation

  1. Rules;
  2. Options;
  3. Validator

Validation rules

  1. Simple an intuitive: *, @, dec, l>=, =#, =, and more...
  2. Can be combined to a simple string: 
    '* @ l>10 !somedomain.com'
  3. Single and multiply (with brackets) rule values:
    • 'l>=10', '<10'
    • '=[13,14]',
      '=#[some_id,another_id]',
      '=[#some_id,somevalue]'

Validation options

  1. Classes - errorClass, disabledClass, processingClass.
  2. Timeouts - timeout and requestTimeout;
  3. Validation event - validateEvent;
  4. Submit button - submitButtonControl, submitButton, submit();
  5. Handler functions - before(), onvalid(), onerror();
  6. AJAX validation options - url, method, data, start(), end();
  7. ... and more ...

Separated validation object

_v_ - is a separate part of tFormer. Used for validation.

// basic usage
_v_('value to validate').validationMethod(options);
_v_('value to validate').validateWithRules(rules_string);

// examples
_v_('').required(); // => false
_v_('i@tjrus.com').validateWithRules('* @'); // => true

View all documented rules and their functions.

Methods

  1. Validation - validateForm, validateField
  2. get/set - get, set, setRules
  3. Submit button control functions - submitButtonOn/Off, processingOn/Off
  4. Errors controlling - errorControl
  5. Form lock/unlock
  6. and main tFormer methods - init, destroy
